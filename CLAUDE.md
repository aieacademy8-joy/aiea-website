\#\# Design Principles

Premium

Minimal

Warm

Educational

Apple-quality

Human-centered

Timeless

Prefer whitespace.

Prefer readability.

\---

\#\# Coding Principles

Keep code modular.

Reuse components.

Avoid duplication.

Keep HTML semantic.

Keep CSS organized.

Keep JavaScript lightweight.

Never break existing pages.

Maintain responsive behavior.

\---

\#\# Navigation Principles

The logo is the Home button.

Do not add a Home menu item.

Always provide intuitive navigation on detail pages.

Prefer breadcrumbs and contextual back links.

\---

\#\# Content Principles

Never invent educational philosophy.

Never rewrite the Academy's core messaging.

Always use the official content library.

Maintain one consistent voice.
