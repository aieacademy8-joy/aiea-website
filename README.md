# AI Explorers Academy™ — Website

A premium single-page site for the AI Literacy ecosystem founded by Miss JOY (Irene Dostov).

## Run locally

It's a static site — any static server works:

```bash
# Python
python3 -m http.server 4321

# or Node
npx serve .
```

Then open http://localhost:4321

## Structure
